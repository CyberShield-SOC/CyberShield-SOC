## Tech Stack

- **Backend:** Python FastAPI
- **Database:** PostgreSQL
- **ML:** Python, Pandas, Scikit-learn
- **Deployment:** Docker, GitHub Actions

## Project Description
CyberShield SOC is a mini-SIEM platform that collects and analyzes
security logs, detects suspicious activity, and displays alerts
on a SOC-style dashboard.

## Features (Planned)
- Log upload and parsing
- Rule-based threat detection
- ML anomaly detection
- Alert dashboard
- Incident tracking and response
- Role-based access control
